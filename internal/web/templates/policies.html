{{template "icon-info"}}How policies compose
A peer applies its policies in order. Import policies only reject, so any one of them can veto a route — they add up to "everything must pass". Export policies only accept, so a route is announced if any one of them permits it, and dropped if none do. Each policy becomes a BIRD function; a peer's filter simply calls them.
